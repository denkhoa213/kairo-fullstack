# FlashLearn

## Mục tiêu

Xây dựng website học Flashcard.

## Công nghệ

- React
- TypeScript
- Vite
- Tailwind
- shadcn/ui
- Express
- MongoDB

## Các chức năng

- Đăng nhập
- Đăng ký
- Dashboard
- Flashcard
- Study Mode
- AI tạo Flashcard
- Spaced Repetition

## Coding Convention

- Dùng TypeScript
- Component PascalCase
- Hooks đặt trong hooks/
- Service xử lý API
- Không viết logic trong UI
- Sử dụng React Query
